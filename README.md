# youtube_api
An airflow DAG that digests Google's YouTube API to explore top videos across 5 countries in a Tableau notebook

https://developers.google.com/youtube/v3


The full interactive version of the Tableau notebook is published on their public site and can be accessed here: https://public.tableau.com/app/profile/lacey.morgan/viz/YouTubeTop50VideostheWeekof84/Dashboard1?publish=yes
